## Projetos em Destaque

### QA Lab Project

Projeto prático de Quality Assurance simulando a atuação de um Analista de Qualidade em uma aplicação web.

O projeto utiliza o site SauceDemo para testes manuais web, com foco em fluxo de login, carrinho, checkout, validações de campos obrigatórios e documentação de bugs.

**Principais entregas:**

- Levantamento e análise de cenários de teste;
- Elaboração de casos de teste estruturados;
- Execução manual de testes funcionais;
- Validação de fluxos de login e checkout;
- Registro e documentação de bugs;
- Análise de impacto de negócio;
- Organização de evidências;
- Versionamento da documentação com Git e GitHub.

[Acessar projeto](https://github.com/brunolopes-ti/qa-lab-project)

### API Testing Project

Projeto prático de testes de API utilizando Postman e a API pública JSONPlaceholder.

O objetivo do projeto é demonstrar validações de requisições HTTP, análise de respostas, códigos de status, payloads JSON e documentação de testes de API.

**Principais entregas:**

- Testes de requisições GET;
- Testes de requisições POST;
- Testes de requisições PUT;
- Testes de requisições DELETE;
- Validação de status code;
- Validação de estrutura JSON;
- Validação de campos retornados na resposta;
- Organização de cenários de teste de API;
- Documentação técnica dos testes executados.

[Acessar projeto](https://github.com/brunolopes-ti/api-tests)

### QA SQL Validation Project

Projeto prático de validação de dados com SQL aplicado à área de Quality Assurance.

O projeto simula um banco de dados de e-commerce utilizando PostgreSQL e DBeaver, com foco em validação de regras de negócio, consistência de dados e consultas utilizadas por QA.

**Principais entregas:**

- Criação de tabelas relacionais;
- Inserção de massa de dados;
- Consultas com SELECT, WHERE e ORDER BY;
- Validações de regras de negócio;
- Consultas com INNER JOIN e LEFT JOIN;
- Validação de consistência entre pedidos e itens;
- Dicionário de dados;
- Cenários de teste SQL;
- Evidências de execução no DBeaver.

[Acessar projeto](https://github.com/brunolopes-ti/qa-sql-validation)
