## Projetos em destaque

### QA Lab Project - Testes Manuais Web

Projeto prático de testes manuais funcionais utilizando o site SauceDemo.

**Objetivo:** demonstrar criação, execução e documentação de casos de teste manuais, com foco em fluxos funcionais de uma aplicação web.

**Principais práticas:**

- Planejamento de cenários de teste;
- Execução de testes funcionais;
- Validação de login, carrinho e checkout;
- Registro de evidências;
- Documentação dos resultados.

Repositório:  
[qa-lab-project](https://github.com/brunolopes-ti/qa-lab-project)

### API Tests - Testes de API com Postman

Projeto prático de testes de API REST utilizando Postman e JSONPlaceholder.

**Objetivo:** demonstrar validação de endpoints, métodos HTTP, status code, payloads JSON e respostas de API.

**Principais práticas:**

- Testes com métodos GET, POST, PUT, PATCH e DELETE;
- Validação de status code;
- Validação de payload JSON;
- Uso de variáveis no Postman;
- Scripts básicos de teste;
- Organização de collection.

Repositório:  
[api-tests](https://github.com/brunolopes-ti/api-tests)

### QA SQL Validation - Validação de Dados com SQL

Projeto prático de validação de dados utilizando PostgreSQL, DBeaver e SQL.

**Objetivo:** demonstrar consultas SQL aplicadas à validação de regras de negócio, consistência de dados e integridade das informações.

**Principais práticas:**

- Criação de tabelas relacionais;
- Inserts de massa de dados;
- Consultas SQL;
- Validação de pedidos, produtos, estoque e usuários;
- Verificação de inconsistências;
- Documentação de evidências.

Repositório:  
[qa-sql-validation](https://github.com/brunolopes-ti/qa-sql-validation)
